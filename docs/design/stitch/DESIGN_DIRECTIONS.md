# Design Direction Comparison

## Scoring criteria

| Criterion | Weight |
|-----------|--------|
| Civic trust / clarity | High |
| Category discoverability | High |
| Mobile responsiveness | High |
| Accessibility (contrast, tap targets) | High |
| Staff + citizen flow consistency | Medium |
| Visual polish for portfolio | Medium |

## Variants

### A — Civic Clarity (WINNER)

- Teal/emerald on slate, light default + dark toggle
- Category cards with colored top borders
- Staff portal badge for role separation
- **Score: 9/10**

### B — Indigo Night

- Current indigo/slate dark-only theme
- Familiar but no light mode, less civic warmth
- **Score: 6/10**

### C — Emerald Gradient

- Green gradients, glass morphism
- Modern but weaker category distinction, contrast risks
- **Score: 7/10**

## Winner rationale

Civic Clarity wins on accessibility (WCAG-friendly contrast in both modes), clear 6-category grid, mobile-first card layout, and distinct staff/citizen visual cues — best fit for a public urban services portal.
